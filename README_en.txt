
ASConsole


1. Summary

ASConsole (Active script console) is an interactive shell for Active Script.
You can use multiple script languages supporting ActiveScript.
ASConsole supports JavaScript, VBScript, Ruby, Perl, Python.


2. Requirements

  Microsoft Windows 7/8/10 + IE11


3. Setup

Please expand the downloaded archived file to any directory.
No special setting is required.

To run the application, double-click the file "ascon.hta" in the expanded folder.

To uninstall the application, delete all the extracted files.
